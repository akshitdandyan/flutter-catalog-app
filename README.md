# App Pulp
### A basic catalog items app.
- Add items to cart
- State management using VelocityX State


#### Requirement for running app
- Android studio setup
- Flutter setup
- After cloning, start app either with debug mode or run 
```
flutter run -d [your-device-ip:port]
```
